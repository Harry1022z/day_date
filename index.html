Ejemplos y Ejercicios

Ejemplo 1: Utilizando DATE_ADD y DAY para calcular días.

SELECT DAY(DATE_ADD('2023-01-01', INTERVAL 10 DAY));

Resultado: 11

Ejemplo 2: Utilizando DATE_FORMAT para dar formato a fechas.

SELECT DATE_FORMAT('2023-12-25', '%d %M %Y');

Resultado: 25 December 2023
